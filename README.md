# VoiceRails Monorepo

This repository contains the VoiceRails v1 platform implementation:

- `packages/connection-layer`: provider abstraction and realtime adapter contracts
- `packages/sdk`: JavaScript SDK
- `packages/react`: React bindings
- `packages/node`: Node webhook utilities
- `packages/cli`: CLI
- `packages/python-sdk`: Python SDK
- `platform/control-plane`: Firebase Functions v2 API
- `platform/telephony-bridge`: Cloud Run Twilio bridge
- `platform/workflow-engine`: workflow interpreter
- `platform/dashboard`: visual builder and console UI
- `infrastructure/terraform`: infrastructure skeleton
