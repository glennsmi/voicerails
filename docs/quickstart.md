# VoiceRails Quickstart

## Prerequisites

- Node.js 20+
- Firebase CLI authenticated to `voicerails8`
- Optional provider secrets configured in environment or Secret Manager

## Install

```bash
npm install
```

## Build

```bash
npm run build
```

## Run Control Plane Locally

```bash
npm run dev:control-plane
```

## Run Dashboard Locally

```bash
npm run dev:dashboard
```
