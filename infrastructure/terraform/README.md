# VoiceRails Terraform Skeleton

This directory is the initial infrastructure layout for the hosted multi-tenant
and customer-dedicated deployment modes described in the platform spec.

Planned module layout:

- `modules/runtime-core`
- `modules/telephony-bridge`
- `modules/event-bus`
- `modules/scheduling`
- `modules/billing-observability`
- `modules/secrets`
- `stacks/hosted-multi-tenant`
- `stacks/customer-dedicated`
